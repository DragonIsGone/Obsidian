### Header comment
Should contain:
- Name of file
- Brief description
- Author (full) name
- Date of creation
- Date last modified
- Version No.
### Naming conventions
- **Variables** use **snake_case**
- **Class** names should use **UpperCammelCase**
- **Constant** name should use **ALL_UPPERCASE**
